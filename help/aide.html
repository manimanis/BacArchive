Utilisation principale
Copier une clé USB
1
Brancher la clé USB du labo
L'application détecte automatiquement la clé et liste les dossiers candidats (format NNNNNN). Si la clé contient plusieurs sous-dossiers de séance, sélectionnez le bon dans la barre latérale.
2
Vérifier Séance et Labo
Confirmez les numéros dans le panneau gauche. Le chemin de destination se met à jour automatiquement : DossierSauvegarde\Séance-NN\Labo-NN.
3
Contrôler la liste des candidats
Vérifiez les statuts : ✓ Présent Absent ⚠ Non-conforme. Tout non-conforme doit être réglé avant de copier — la copie sera bloquée.
4
Cliquer « Copier & Vérifier (MD5) »
Chaque fichier est copié puis vérifié par empreinte MD5. En cas d'erreur, la copie est relancée automatiquement. La destination reste intacte si la copie échoue.

Référence
Statuts des candidats
StatutCondition de détectionCopieGrille Excel
✓ Présent Au moins 1 fichier dans le dossier (recherche récursive) Intégrale + vérification MD5 Ligne normale, saisie de note
● Absent Aucun fichier + sous-dossier Absent (ou Absente, Abs…) Dossier copié tel quel Ligne grisée, note 88,88
⚠ Non-conforme Dossier totalement vide — aucun fichier, aucun sous-dossier Copie bloquée —
⛔ Fraude Détecté par BacBackup Copié avec signalement —
⚠️
Dossier Non-conforme : un dossier totalement vide est ambigu — impossible de savoir si le candidat a travaillé ou s'il était absent sans documentation. La copie est intégralement bloquée jusqu'à résolution. Pour documenter une absence : créer un sous-dossier Absent vide dans le dossier du candidat sur la clé USB, puis appuyer sur F5.
